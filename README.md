# SPA-VanillaJs

## SPA приложение на основе web компонентов.

## Как пользоваться

npm install

npm install -g json-server

npm install -g http-server-spa

## Запуск локального REST API сервера

npm run server

## Запуск приложения

npm run start

## Команды для сборки дев и прод версий

npm run build

npm run ibuild-prod
